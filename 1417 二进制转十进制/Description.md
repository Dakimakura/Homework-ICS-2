# 二进制转十进制

## Description

在计算机中整数通常以二进制的方式存储，但在生活中一般使用十进制，请写一个程序将二进制数转换为十进制数。


## Input

一个长度不大于32的01串，输入的串，不带\n，直接以文件结束符作为结尾

## Output

这个01串所代表的二进制数对应的十进制数
